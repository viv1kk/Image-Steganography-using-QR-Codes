Image Steganography

    The word Steganography is derived from two Greek words- 'stegos' meaning 'to cover' and 'grayfia', meaning 'writing', thus translating to 'covered writing', or 'hidden writing'

    Steganography in images is a method of hiding or concealing an image / data inside an image without leaving any humanly perceivable trace of the concealed image / data on the base image.

    This is a Web based UI for demonstrating the implementation of Image Steganography. There are 2 operations Encode and Decode, Encode will conceal an image or a text data inside a base image and save the stego-image, and Decode will extract the hidden image inside/text from the stego-image.
